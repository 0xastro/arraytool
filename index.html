Welcome to Arraytool’s documentation!¶

Release:0.0.0
Date:June 14, 2011

Arraytool is a Python library for analysis and design of phased array antennas. For general information about Arraytool, see the project repository and blog.

These documentation pages include general information as well as docstring listing with use of examples. For quick access to the docstrings of individual functions, use the index listing.
